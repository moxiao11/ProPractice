# 程序设计实践课程

> 2026 春季学期 · 计算机相关专业

### 课程介绍

《程序设计实践》是一门以动手编程能力培养为核心的实践型课程，旨在帮助学生将程序设计基础知识应用于真实问题的解决，提升算法思维、系统设计能力与工程实现能力。

### 你将获得什么

通过本课程的学习，学生将能够：

- 熟练掌握一门主流编程语言
- 理解并应用常见数据结构与算法思想
- 独立完成小型规模程序设计项目
- 具备调试、测试与代码优化能力
- 形成良好的代码风格与工程意识

### 学习建议

1. 先阅读本页“课程安排”，明确每阶段目标。
2. 按顺序完成每次实验，确保每一步都能独立跑通。
3. 遇到问题先复现并记录，再查文档、提问或讨论。
