# 图形化界面

### 是什么

图形化界面（Graphical User Interface）就是你通过图标、窗口、按钮、菜单等可视化元素来操作电脑或软件的界面，而不是靠输入一行行命令（CLI）。

两者的区别可参考：<a href="#/assets/content/knowledge/CLI.md">命令行(CLI)</a>

### 典型组成部分

常见元素包括：窗口、按钮、菜单、文本框、复选框和图标等。

### 如何实现GUI

不同的编程语言有GUI框架

| 语言 | GUI框架 |
| --- | --- |
| C++ | `QT` / `wxWidgets` / `MFC` |
| Python | `Tkinter` / `PyQt` / `Kivy` |
| Java | `Swing` / `JavaFX` |
