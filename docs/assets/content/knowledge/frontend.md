# 前端

---

### 是什么

前端（Frontend），简单说，就是用户在网页或 App 上“看得见、点得到、能交互的那一部分”。

### 怎么实现

前端有三剑客，html,css,javascript,下面的图很精确的展示了这三门语言的用处


![](../../img/1.png)

* html 搭建骨架，按钮，搜索框
* css 用于美化你的网页，带了色彩
* js用于实现按下这个按钮有了跳转功能
