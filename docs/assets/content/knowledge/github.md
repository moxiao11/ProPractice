# github

### 是什么

GitHub 是一个基于 Git 版本控制系统 的代码托管与协作平台，用于存储、管理和协作开发软件项目。

###  核心功能
* 代码托管

开发者可以把本地项目上传到 GitHub 服务器进行保存和管理。

* 版本控制

基于 Git，可以：

记录每一次修改

回退到历史版本

比较不同版本之间的差异

* 团队协作

多人同时开发同一个项目

分支（Branch）管理不同功能

Pull Request（PR）代码合并

Code Review 代码审查

* 开源社区

GitHub 是全球最大的开源社区，很多知名项目都托管在上面，例如：

Microsoft

OpenAI

Google

以及开源项目：

Linux

TensorFlow

React
