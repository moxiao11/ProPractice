# 写在前面

任何一门课程的改革，都不可能是一条平直而顺畅的道路。
它几乎注定要经历质疑、吐槽、抱怨，甚至误解。
也许在若干年后，这些细节都会被时间抹平，只剩下几句在学长学弟之间口口相传的话：

* 某某学长把这门课搞难了。
* 某某学长写的代码真不怎么样。
* 某某学长改的就像个笑话。

那句话其实说的很对，学生在成绩和能力之间选择成绩，老师在改革和维持现状间选择后者，但总有人要来干这件事，其实也为了更多普通的人，如果你去问各种AI，对于普通人来说最好的专业是什么，那么它一定回答你，是计算机科学与技术，我不是高中接触信奥赛的OI，也不是那种随口就说出各种专业名词，什么token，什么前后端。

大一的我连翻墙都是个问题，但是我想说的是我用自己的来路证明了，关关难过关关过，前路漫漫亦灿灿，不断反复锤炼代码，不断接触更多新的概念，不断理解专业课知识，或许脑中仍然迷雾重重，但未来的你一定会感谢现在的自己。
