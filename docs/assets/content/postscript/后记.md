# 后记

> 山林从不向四季起誓，枯荣随缘，因此季节决定生死落叶

####  学长的话

心态是最好的风水，一定要允许一切的发生。学长把自己的很多所见都写成文档，只是为了帮助曾经一样的自己，听着别人说的各种高大尚的词汇，什么数据库，什么Token，什么前端后端，后来才发现，学计算机就和浙大翁凯老师说的那样:"所有的知识，只是我现在不知道，未来终有一天我会把那个底层弄得清清楚楚。"也请现在的你加油，为了那个年少不服输的自己。

说回这门课，承蒙岁月厚爱，让我得以将许多时间与心力，倾注于这门课程之中。其间或有遗憾，亦不乏精彩；纵使仍存疑惑，却也在摸索中逐渐走出了自己的节奏与风格。学习本就不是一条笔直的路，重要的并非事事笃定，而是在不确定中仍愿意前行。

当你读到这段文字时，或许是一段旅程的起点，或许已临近尾声。但真正不变的，从来不是课程的进度，也不是一次次作业与考试，而是你一路走来所累积的思考、选择与成长。那些看似零散的努力，终会在某个时刻，悄然汇成属于你自己的答案。

这一程，始于春日，终于炎日。愿你在此间学到的不只是知识与方法，更有面对未知时的从容与勇气。前路仍长，山海未歇，愿你此去心有所向，步履坚定。

####  为什么你觉得每条路总是很累？
    
因为父母受限于时代，也受限于他们自身的能力，他们无法替你看清世界，更无法在你迟疑时告诉你哪一条路才算正确。站在人生的岔路口，你只能独自做选择；跌倒了，也只能自己站起来。你说不清自己真正想要什么，也分不明喜欢与擅长，只是被时间推着往前走。我们总以为远方更重要，于是不断加速，生怕一停下就被落在身后。黑暗里没有路标，你只能不断试错，而每一次错误的代价，都重到必须亲自承担。

但你要告诉自己：不要后悔任何已经做出的决定。我从不怀疑当时的判断，事与愿违，并不是选择出了问题，而是那时的我，还撑不起想要的结果。人生不可能步步正确，也没必要频频回头审判自己。认知有限，选择有限，后果也只能自己承担。

所以坦然一点，为自己的认知买单。不要假装当初可以做得更好，也不必用现在的眼光去否定过去的自己。若是重来一次，以当时的阅历与能力，我依然会走到同样的路口，也依然会踩进同样的坑。区别只在于，这一次，我更清楚自己为什么会摔倒，也更明白该如何继续往前走。
