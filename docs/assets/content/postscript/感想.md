# 学习

于2026年3月8日突发感想写下此文

### 一段小背景

说实话，最近在学习过程中看的非常多的课，强度都挺大，看着其他人网上写的题解，似乎感觉很多东西别人就是看了视频就该会的样子，大一的时候以为MIT的missing lecture的知识密度就是巅峰了，没想到还有很多强度很高的课。这个过程中我其实特别特别迷茫，配置一个xv6操作系统环境，参考了特别多的文章，似乎没有什么得天独厚，一会儿是ubuntu的版本不对，然后版本对了又出现vmware黑屏的情况，光是配环境就等那个加载进度条，快搞了一天。开始写代码又会发现，这东西我上课教了吗？c语言我好像会，但是这写的是c语言吗，一看前置课程，CSAPP里面讲了，又跑去看那个课，看了半天前置课，发现自己c语言差了点，查了半天资料。感觉整个过程像在套娃，东拼西凑自己的知识体系，做题的过程十分煎熬，感觉没有一点思路。

### 反思

后来反思这个问题，其实一切的根源就在于很多东西我们学了就忘，只把它当做考试的工具，当我后面再要用起来的时候会发现，一切早已烟消云散了。不禁发问：那刷这一门门课还有意义吗？答案当然是肯定的，当你上完一门课，可能只记住了30%，需要不断地重复，即使这个过程很煎熬，可能花一天，一个星期，一个月重复一个东西，在别人看来这就是一个很简单的事情，我们像傻子一样重复了太多次。这个推广之上，其实更多涉及到自己的价值，后来我也逐渐的赞同了支老师说的一句话，vector已经很成熟了，为什么我们还要用数组呢? sort很方便，为什么我们还要一次次的手写排序算法呢？一件事情本身是没有意义的，但是深度理解难懂的知识，锤炼自己的心性，我们反而将其赋予了价值。

### 建议

所以，不必在意别人的眼光，或许我们干了很多没有意义的事情，但是成长的过程独属于你，或许你是看着题解去一步一步敲的代码，但每一步都是你的见识，每一行都是你的成长。
