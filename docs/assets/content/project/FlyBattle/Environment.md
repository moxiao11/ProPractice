# 飞机大战环境配置

### 配置环境
 
- C++：CMake + C++ 编译器（MSVC / GCC / Clang）
- **Node.js**：v14 或以上

对于cmake和node的安装其实依旧推荐使用<a href="#/assets/content/knowledge/Scoop.md">scoop</a>，当然去官方安装也使可以的，对于C++大部分同学上学期应该已经安装过g++。

这里温馨提醒一下：装CLion的同学可能没有安装过g++和使用过g++的指令，还是安装后熟悉一下<a href="#/assets/content/knowledge/g++.md">g++</a>，同时全过程代码由Vscode编写，但是学长也提供了shell和bat脚本，理论上当环境足够的时候，一键运行也是可以的。
